## Usage

Insert the code below `between <body> and </body>`.

```html
<script src="canvas-ball.js"></script>
```

Strongly suggest to insert before the tag `</body>`, as the following:

```html
<html>
<head>
	...
</head>
<body>
	...
	...
	<script src="canvas-ball.js"></script>
</body>
</html>
```

Then ok! `Please do not add the code in the <head> </head>`.
